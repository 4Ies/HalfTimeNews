This application is been developed as a personal project for fun and because
of the need of having something easy to use to remain updated on the news of 
the world: I implemented this app using python and Ubuntu (deb package to be 
compiled) so venv was used to set up a virtual enviroment. The aim of this 
project is to have news grouped by the same argument with the aid of AI models
used offline and using multiple customizable fonts.

\subsection{Needed Libraries and Software}
To run a working setup like the one that i use (on an Ubuntu machine) it is
necessary to have python using venv, here the list of libraries used:
\begin{itemize}
    \item \textbf{feedparser} - a useful library to browse all the attributes 
    of RSS elements
    \item \textbf{ddgs} - the duckduckgo search library
    \item \textbf{PyQt6} - to manage UI
    \item \textbf{sentence-transformer} - to group related news with aid of ai
    \item \textbf{APScheduler} - to do periodic fetches of news
\end{itemize}
All those libraries can be installed with the command pip install libraryName
