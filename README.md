# ◈ Half Time News

A personal news aggregator that uses rss and clustering to have all news in the same place

## Developement and Prerequisites
The program is composed by a main python file and it has been implemented on ubuntu using venv. To clone this repository and use it it is sufficient to execute this command in terminal inside the desired folder:

```bash
git clone https://github.com/yourname/newsagg.git
cd newsagg
python3 -m venv venv
source venv/bin/activate
```
Notice that several python libraries were used and can be installed using `pip install` command and the requirements text:
```bash
pip install -r requirements.txt
```

To finally run the program, simply:

```bash
python main.py
```
